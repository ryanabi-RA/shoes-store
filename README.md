### How to run in localhost

1. clone project in directoy pc
```
git clone https://github.com/ryanabi-RA/shoes-store.git
```
2. install dependencies
```
npm install
```
3. run poject
```
yarn dev
```
4. view project in browser
```
http://localhost:3000
```
